# projekt_informatyka_2025-2026cz2
Autor: Jakub Mitura s204082
Celem mojego projektu było stworzenie prostej i interaktywnej symulacji komputerowej, która pokazuje, jak wygląda podstawowy proces wytwarzania prądu w elektrowni węglowej przy użyciu pary. W aplikacji można zobaczyć kolejne etapy od zasobnika paliwa, przez piec i kocioł, aż po prądnicę, tak żeby dało się łatwo prześledzić jak wszystkie te elementy współpracują ze sobą, prowadząc ostatecznie do produkcji energii elektrycznej.

Projekt spełnia wszystkie wymagania postawione przez prowadzącego: zawiera wizualizację przepływów, sześć zbiorników połączonych rurami, możliwość przełączania między trybem automatycznym i manualnym sterowania procesem, dwa oddzielne ekrany (jeden do sterowania, drugi do wizualizacji), a także implementację zbiorników jako obiektów. Do stworzenia aplikacji wykorzystałam biblioteki PyQt5 oraz pyqtgraph.
